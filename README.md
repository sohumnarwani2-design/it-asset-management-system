## 💡 Problem Statement

Most IT rental businesses still rely on Excel sheets to manage inventory, track rentals, and monitor returns. This process is time-consuming, error-prone, and difficult to scale. Additionally, existing inventory management solutions are often expensive and not tailored to smaller businesses.

## 🚀 Solution

This project was developed to automate the entire asset management workflow — including inventory tracking, rental lifecycle management, and maintenance logging — through a simple and cost-effective web-based system. It helps reduce manual effort, improve accuracy, and save time for business operations.

## 🛠️ Tech Stack

* **Backend:** Flask (Python) for handling application logic and APIs
* **Database:** PostgreSQL (hosted on Supabase) for scalable data management
* **Frontend:** HTML and CSS for user interface design

The system is designed with a modular architecture, enabling easy scalability and future integration of advanced features such as analytics and AI-based predictions.
